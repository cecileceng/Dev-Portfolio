Alexa Adventures RPG

Created an Amazon Echo text adventure powered by Alexa Voice Service in 24 hours for AngelHack Austin 2016. I assembled a team of friends, created the decision tree used for the text adventure, created original story content, input that content into our TSV parsing library using Python, and helped research the limitations of the Alexa Voice Service API.  It was our first hackathon and our project won Best Use of Alexa Voice Service.

Watch our demo video here.

Troll Bridge

For my first project with the Coding Boot Camp, our team of four designed an app that aggregates social media accounts and used HPE Haven’s Sentiment Analysis API to determine if someone is overall positive or negative in order to show their “troll footprint”. I implemented Single Sign-On using OAuth in order to access the Twitter API and receive tweet message data. From there I learned how to use RegEx in order to clean up the data and remove characters that the Sentiment Analysis couldn’t process.

See it in action here or view the code here.

Disney Princess Hangman

A command line hangman game created during my time at the Coding Boot Camp. I utilized Node.js and object-oriented programming to create a command line utility to allow users to play hangman inside their terminal. I wrote a dictionary parsing algorithm, and went beyond the requirements by writing a complex RegEx to handle character visibility and parse user inputs.
